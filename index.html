Proyectos

En esta sección listaré algunos proyectos propios en los que estuve trabajando.

Proyecto 1

La idea de esta pequeña aplicación web, es servir de estación de ensayo para aquellas personas que estén iniciando o tengan cierto conocimiento en guitarra o bajo. El usuario puede elegir en qué tonalidad trabajar, y la aplicación mostrará dinámicamente los grados correspondientes a dicha tonalidad, las tríadas, y simulará el mástil del instrumento seleccionado. A su vez, el usuario podrá tocar las notas en el mástil y el programa reproducirá el sonido de dicha nota. La aplicación cuenta también con un metrónomo y un temporizador, para controlar el tiempo que se desea practicar.

El proyecto no está terminado aún. Tengo varias funcionalidades que quiero agregar, y algunas cosas que me gustaría mejorar. De todos modos, se puede utilizar clickeando aquí y es posible descargarse el repositorio desde GitHub para ser utilizada.

Proyecto 2

En este caso, quise crear un videojuego web de gestión de recursos. En él, el jugador deberá administrar su dinero para cumplir los pagos que periódicamente le irán cobrando, cada vez con más intereses. Para ello, deberá hacer buen uso de los recursos del terreno de juego (generado de forma aleatoria en cada partida) para generar activos que le permitan cumplir con los pagos antes mencionados. Podrá crear aserraderos, granjas, minas, puertos, tiendas y fábricas. Cada construcción tendrá sus costos y sus beneficios.

El juego se encuentra en desarrollo. Tengo que terminar algunos puntos, y mejorar otros. De todos modos, ya es jugable y permite entretenerse buen rato. Todo el proyecto es de mi autoría, incluído los gráficos. Como el resto, podrán encontrar el repositorio del proyecto en GitHub para lo que deseen. Click aquí si deseas probar el juego.
