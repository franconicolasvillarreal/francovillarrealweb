Gestor de finanzas personales

Gestor de finanzas personales


Creé esta aplicación para decirle adiós a mis planillas de Excel y gestionar mis finanzas personales de forma más práctica. Es posible agregar distintos tipos de movimientos, visualizarlos, filtrarlos y ordenarlos. A través de un sistema de cómputo de datos, es posible llevar el control del flujo del dinero a cada momento, además de poder proyectar para los meses subsiguientes. Por otra parte, utilicé una API para obtener la cotización de algunos valores y verlos en pantalla en todo momento. La aplicación posee el backend desarrollado en NodeJS y el frontend en VueJS. Para la persistencia de datos opté inicialmente por MySQL pero luego migré a archivos .JSON. De esta manera, cualquier persona podría instalar la app, ejecutarla y utilizarla en su computadora sin necesidad de configurar una base de datos. Si bien la app no está terminada, ya es utilizable. Podrá encontrar el repositorio del backend aquí y del frontend aquí.

Videojuego de gestión de recursos

Videojuego de gestión de recursos


En este caso, quise crear un videojuego web de gestión de recursos. En él, el jugador deberá administrar su dinero para cumplir los pagos que periódicamente se le irán ejecutando, cada vez con más intereses. Para ello, deberá hacer buen uso de los recursos del terreno de juego (generado de forma aleatoria en cada partida) para generar activos que le permitan cumplir con los pagos ya mencionados. Podrá crear aserraderos, granjas, minas, puertos, tiendas y fábricas. Cada construcción tendrá sus costos y sus beneficios. El juego se encuentra en desarrollo. Tengo que terminar algunos puntos, y mejorar otros. De todos modos, ya es jugable y permite entretenerse buen rato. Todo el proyecto es de mi autoría, incluído los gráficos. Como el resto, podrán encontrar el repositorio del proyecto en GitHub para lo que deseen. Click aquí si deseas probar el juego.
